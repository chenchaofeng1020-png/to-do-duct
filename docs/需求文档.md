# 每日待办与备忘卡片化 iOS 需求文档（本地存储）

## 存储与部署策略
- 纯本地数据存储即可完整运行与发布，无需任何后端部署。
- 默认使用 Core Data（或 SQLite）保存数据，遵循系统沙盒。
- 备份与恢复：
  - 随设备备份（iCloud 设备备份或 Finder/iTunes 备份）自动包含 App 数据。
  - 可选启用 iCloud/CloudKit 做跨设备同步（不依赖自建服务器）。
- 隐私与安全：数据不出端，仅在用户开启同步时走 Apple 官方云服务。

## 产品愿景
- 以“按天组织 + 任务延续”构建轻量待办：每天一张卡片记录当天任务；同一任务可跨天延续并直观呈现“持续天数”。
- 视觉高对比、圆角与胶囊元素；顺滑动效与简洁交互；专注当天完成与习惯培养。

## 核心概念
- 待办卡片（Day Card）：自然日容器，标题为当天日期（如“2026年1月8日”）。
- 待办事项（Todo Item）：卡片内任务条目，可勾选完成。
- 持续天数图标（数字胶囊）：事项前的紫色数字，表示“同一件事已连续出现的天数”。
- 继续一天（Continue One Day）：将事项延续到目标日期卡片，形成同一“持续链”。
- 备忘卡片（Memo Card）：在“备忘录”页，单条备忘以卡片形式呈现，参考 flomo 的卡片化信息流。

## 关键功能（MVP）
- 新的一天按钮：点击创建当天卡片（若已存在则定位到该卡片），自动以系统日期命名。
- 卡片内新增待办：卡片底部输入框回车添加；空输入拦截、去除空格；新建事项默认未完成。
- 继续一天：
  - 在事项的操作菜单中点击“继续一天”。
  - 弹出选择器：默认建议“下一天”（若不存在则可一并创建），也可选择任意后续日期卡片；默认不允许过去日期（设置中可开启高级模式）。
  - 确认后，在目标卡片中新建同标题事项，与源事项建立同一“持续链”。
  - 显示数字胶囊：链上各事项显示其链位序（1、2、3…）；为简洁起见，链位序为 1 时可隐藏数字，仅从 2 开始显示。
- 事项操作：勾选完成、编辑标题、删除、拖拽排序；删除会同步影响链的显示逻辑。
- 搜索：顶部搜索框全局搜索卡片与事项标题；中文分词优先；结果点击定位到对应卡片与事项。
- 备忘录（卡片化）：
  - 新建备忘：输入后生成一张“备忘卡片”，立即显示在列表中（参考 flomo）。
  - 卡片内容：文本（支持换行与基础高亮）、时间戳、可选标签/颜色；支持置顶、归档、编辑与删除。
  - 列表：按时间倒序的卡片流；轻触进入详情或内联编辑。
- 设置：日期显示样式、延续规则（是否允许过去日期）、提醒偏好、视觉主题与暗色模式。

## 持续链与数字胶囊显示规则
- 链的生成：首次“继续一天”时为源事项与目标事项分配同一个 seriesId；源事项 chainIndex 设为 1（可显示/可隐藏），目标事项 chainIndex = 2。
- 链的增长：当从链的最新事项继续到更晚日期时，目标事项 chainIndex = 最新 chainIndex + 1。
- 显示原则：
  - 每个链内事项显示其 chainIndex；当 chainIndex ≥ 2 时以紫色胶囊展示数字。
  - 若用户从中间节点继续（非最新）或跨多个日期插入，系统自动以日期顺序重排并重算 chainIndex，确保数字与时间顺序一致。
- 删除与插入：
  - 删除链中任一事项：其后续事项的 chainIndex 依日期顺序自动重算（例如原 1→2→3，删除 2 后变为 1→2）。
  - 插入到更早日期：默认不允许；若在设置中开启高级选项，允许插入则按日期重算 chainIndex 并弹出提示。
- 合并链：仅当手动“合并链”操作时才会统一 seriesId 并重算；默认以 seriesId 为准。

## 交互流程
- 创建当天卡片：点击“新的一天”→新卡片置顶→自动聚焦输入框。
- 添加事项：输入→回车→插入到卡片顶部→轻触反馈。
- 继续一天：长按/更多菜单→继续一天→选择目标日期（默认下一天）→目标卡片生成事项并显示数字胶囊。
- 勾选完成：复选框切换→事项置灰/可移至“已完成”分区（可选）→链数字不随完成状态改变。
- 搜索与定位：输入→过滤→点击结果跳转并高亮对应事项。
- 删除与撤销：滑动删除→显示“撤销”吐司→5 秒内可恢复并回滚链重算。
- 备忘页：输入→生成备忘卡片→卡片化信息流展示→编辑/置顶/归档。

## 信息架构与导航
- 底部标签栏：
  - 待办事项（首页）：日卡片列表、搜索、创建新的一天、继续一天。
  - 备忘录：独立卡片化笔记列表与编辑器。
- 设置：允许选择任意日期延续、显示 1 的胶囊与否、提醒偏好、主题与暗色模式。
- 个人入口：顶部左侧圆形头像入口，进入设置与资料。

## UI 设计规范（参考第二张图风格 + flomo 卡片化）
- 颜色：
  - 主色（强调）：荧光黄（Neon Lime，#EFFF59~#F3FF57）用于主按钮与关键强调卡片。
  - 次强调：紫色（#7C6BF2）用于数字胶囊与选中态。
  - 文本：深黑/墨色；次级文本灰色；背景为柔和米白渐变。
- 组件：
  - 新的一天按钮：大圆角胶囊、主色填充、加号图标，按下态略微缩放与暗化。
  - 搜索框：圆角、浅灰背景、左侧搜索图标、占位“搜索…”。
  - 待办卡片：日期标题、事项列表、底部输入框；卡片阴影轻柔。
  - 事项行：左侧复选框，标题旁显示紫色数字胶囊（chainIndex≥2）；右侧“更多”菜单含“继续一天”。
  - 备忘卡片：圆角卡片，内容文本、时间戳、可选标签/颜色、置顶标识；卡片化信息流。
  - 底部导航：高对比图标，当前标签高亮；整体悬浮视觉。
- 动效与反馈：创建/延续/勾选/删除/新建备忘均有统一动效与 Haptics；列表滚动与定位平滑。
- 暗色模式：主色适配暗底；文本对比满足 WCAG AA；阴影在暗色模式改用轻微发光。

## 数据模型（本地）
- DailyCard
  - id: UUID
  - date: Date（仅日期，遵循系统时区与区域格式）
  - createdAt: Date
  - items: [TodoItem]
- TodoItem
  - id: UUID
  - cardId: UUID
  - title: String
  - isDone: Bool
  - createdAt: Date
  - completedAt: Date?
  - orderIndex: Int
  - seriesId: UUID?（同一持续链的标识）
  - chainIndex: Int?（该事项在链中的序号；首项为 1）
- MemoCard
  - id: UUID
  - content: String
  - createdAt: Date
  - updatedAt: Date?
  - tags: [String]?
  - color: String?
  - pinned: Bool?
  - archived: Bool?
- 可选辅助：ContinuationSeries（id、startDate、lastDate、titleSnapshot）；初版可不落库，按需要动态计算。

## 业务与算法细节
- 创建链：首次继续时生成 seriesId，并为源项与新项设定 chainIndex（1 与 2）。
- 增长链：以目标日期是否晚于链最后日期判断；若不是最后节点，插入后按日期全量重算 chainIndex。
- 重算策略：按 seriesId 聚合、按 date 排序，为每项赋 1..N 的 chainIndex；UI 端使用 chainIndex ≥ 2 显示胶囊。
- 性能：链重算在持久化层异步执行；UI 端使用快照刷新，避免主线程阻塞。

## 技术与架构
- 框架：SwiftUI + MVVM；Combine 处理状态与数据流；支持 @Observable / ObservableObject。
- 持久化：Core Data（或 SQLite），提供 seriesId 与 chainIndex 字段；重算任务采用后台队列。
- 模块划分：
  - TodayCards（含继续一天）、Memo（卡片化）、Search
  - Shared：DesignSystem、Persistence、Haptics
- 测试：
  - 单元测试：链创建、插入、删除后的 chainIndex 重算；搜索与排序；备忘卡片 CRUD。
  - UI 测试：继续一天选择器、跨日创建、撤销与恢复路径、备忘卡片创建与编辑。

## 通知与提醒（扩展）
- 卡片级提醒：为每天设定提醒时间（如 21:00 回顾当日）；文案可显示今天链的新增项数。
- 事项提醒（后续）：单事项到期提醒与延续建议。
- 技术：UNUserNotificationCenter，首次启动引导申请权限。

## 无障碍与本地化
- 动态字体、VoiceOver 标签与可操作控件顺序；触控目标≥44pt。
- 本地化：中文简体优先；日期格式遵循系统区域设置；易扩展英文。

## 性能与质量
- 冷启动 < 400ms；滚动与创建交互 60fps。
- 列表虚拟化与惰性加载；持久化操作异步化。
- Crash-Free Sessions ≥ 99.9%。

## 隐私与安全
- 数据仅存储在本地或用户 iCloud；不采集敏感信息。
- 不记录明文隐私；遵循系统权限申请最小化。

## 边界与异常
- 同日重复创建：阻止并定位已有当天卡片。
- 选择过去日期：默认禁止；若开启高级选项，允许但需重算链并弹出提示。
- 删除链项：自动重算后续 chainIndex；提供撤销。
- 标题修改：链内项允许独立修改标题但仍共享 seriesId；设置中可启用“标题联动”。
- 超长文本：折叠展示并支持展开。

## 验收标准
- 纯本地运行，无需后端部署；数据随设备备份。
- 我可以创建当天卡片、添加任务、发起“继续一天”并看到正确的数字胶囊。
- 删除或插入链项后，数字按日期顺序正确重算。
- 我可以新建备忘并以卡片形式显示、编辑、置顶与归档。
- 搜索可定位待办与备忘；暗色模式与无障碍可用；交互顺滑、视觉符合参考风格。
